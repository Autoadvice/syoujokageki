动画12集中明明两人已经创造出了新的剧情，找到了属于自己的starlight，剧场版一开始却还是出现了大量意向化的悲剧象征，说明剧场版并非动画的剧情延续，而是对于动画中逻辑细节的补充，对于众人遗憾的补完，对于人物命运的深挖。

一个炼钢池里，滚烫的铁水沸腾着，这是上一个电弧炼钢的下一个工序，象征舞台少女的自我矛盾已经临近爆发。

一片沙漠中，长颈鹿焦急的流着汗赶路，一边念叨自己可能赶不上，似乎生怕错过了什么好戏，这样焦急的长颈鹿第一次出现，代表剧场版的剧情是对于动画剧情的加强，而明明长期居住在地下剧场的长颈鹿，汗流浃背，似乎要前往远方似的慌忙赶路，象征剧场版的主要舞台不在音乐学园的地下，而是更加广阔的现实世界，长颈鹿不惜舟车劳顿都要前往的全新舞台，象征舞台少女们的前路十分遥远，并且和学校选拔的对比如地下剧场和沙漠的对比一样明显。

华恋躺在沙漠网格当中，被约束在格子内，身边有一些positionzero的褪色模型，代表旧有的舞台虽然被打破，但是冥冥之中还有其他因素将两人把玩于命运的手掌心。

华恋一只手拿着纽扣，代表她目前是没有登上舞台的状态，但是穿着舞台服装，就说明现在的华恋是战败的状态，或者是在没有罩衣的状态下展开revue，这一幕的意向化意义会在等会儿的剧情中交代。

铁水倒入铸造炉，看上去只是前面两个象征镜头的延续，其实还代表着舞台少女从可以自由表达的学校，走向阶级和结构固化的社会，被塑性成不同的形状，就如这铁水一般。

两片幕布中间，摄像机掉落，象征传统意义上的观众已经消失，砸坏舞台设备，代表固定剧情的装置已经损坏，剧情将脱离既定剧本。

东京塔上的色泽也出现了电子失真般的崩坏，象征学校的演出虽然结束，但是原来的视角和认知已经过时，接受不了属于自己的那个乌托邦的舞台已经结束，让舞台少女大跌眼镜的狂野新舞台出现，甚至让自己认知中象征闪耀和权力的东京塔都出现了裂缝，也出现了纯洁信仰中被污染的坏块，一切镜头语言，都是对于动画版剧情的大刀阔斧的革命。

台词：“你的下一次出场，又在第几幕”出现。
看似没头没脑的、缺少语境的台词，其实和荒谬的画面非常合衬，对于旧规则的打破，和对于未来的迷茫，完美配合在了一起。这一句台词同时也是贯穿全剧的一句话，代表着中心思想。

镜头旋转，闪耀着光芒的东京塔是斜着的，灯光变幻，除了观看台的部分以外，其他塔身结构都关上了灯光，象征着视角变化之后，才发现闪耀着的只有自己所处的地方，认为闪耀的人，只有自己和自己的同伴，除了自己以外的广大地区，对于舞台的认识，是一片混沌和黑暗的，表现出了舞台艺术的接纳程度不高，群众对于舞台少女们要表达的思想毫无认知。

长颈鹿很害怕自己错过一场好戏，但是在一座壮硕摘星塔面前感叹好戏才刚刚开始。才刚刚开始这句话象征着进入全新舞台的众人面对的是全新的挑战，是对自己旧有认知的彻底打破，是对观众来说的更有冲击力的剧情，塔身变形，象征所追寻的事物不再单纯，而摘星之路又要重新走过。

神乐光牵动绳索，控制武器，将高塔内壁给打破，象征着已经打破剧本的束缚，同时交代了这场revue的主角是光恋两人，将她们隔开的高塔已经坍塌，象征象牙塔的倒塌，和残酷现实，也就是沙漠，的出现 。

华恋迷茫的从沙漠中移动到冒着硝烟的塔身上，狼烟弥漫，华恋和神乐光兵戎相见，顺滑的3D动画和磅礴的音乐将气氛渲染到了小高潮，难道，这又是一场激烈的对决吗？

神乐光背后冲起滔天狼烟，将头发吹的竖直（这里的光好漂亮），华恋迷茫的从大漠中转移到战场上，说明命运的变化是在没有准备的情况下就发生了的，硝烟将头发扬起象征命运让人身不由己，同时也表现出了神乐光的杀气。

塔身灯光关闭，华恋拿剑刺去，却被光徒手接下，象征最残酷的战争在黑暗中就开始了，而这一幕具有观赏性和冲击性的场景被两人身后的霓虹灯照亮，象征只有高潮的镜头，才被舞台点亮，被众人看见，而黑暗处的战斗，不为人知。剧场版中有多处用黑暗配合残酷的打斗场景、用光芒配合尘埃落定后的冲击性镜头的表现手法，可以多留意。

这种表现手法和动画版最大不同是仅仅展现战斗结果，而忽略战斗过程，表现出新舞台对于结果的重视，对于舞台少女们个人情感的忽视，而舞台少女奋力地表演，就多了一层悲壮的色彩。在刚才一幕中，观众被渲染起来的情绪，顷刻间便消散，表现了剧场版主要思想之一——观众对于复杂而亢长表演的轻视。

塔身玻璃碎裂，无数position zero的模型腾空而起，象征闪耀舞台的鲜艳外衣，在此刻被打破，露出残酷的真相。

一片断壁残垣之中，模型贯穿观景台，直冲云霄，象征舞台并非由少数演员所垄断，而是无数演员在一片荒芜中构成，无数的主角和配角，像画面中的模型一样，鱼贯而入、又轰然而出，在观众眼里甚至停留不了多久，但这些闪耀的文化瑰宝，会化作星光，永远留在空中，供有能力欣赏的人在千百年后继续瞻仰。

荒芜的背景表现出旧物台道具已经被损坏，象征……自己想。

starlight是注定会离别的悲剧，这句话，在这样镜头语言的配合之下，多了一层新的朦胧的意思，象征着比两人分别的更重大危机即将到来。

华恋看着数量庞大的模型飞上天空，代表演员们在面对自己闪耀流逝之时的迷茫无助和思考，模型是灰色的，代表过去的闪耀已经一文不值。

闪耀过后，一片残破景象出现，象征表演结束后的幻想破灭，以及艺术被观众淡忘后无人问津的惨状。

东京塔被拦腰截断，在一片残破之中，华恋依旧躺在沙漠的网格当中，象征这一切的荣华，都和演员没有太多关系，繁华落幕依然是囚禁自己的枷锁。

神乐光站在一堆模型上，身旁插着一只粉红色的模型，上面挂着华恋掉落的罩衣，正好处于阳光的光芒中心，说出命运会改变的台词。象征光虽然知道舞台的虚假，但是却把最后的真实，留给华恋，希望华恋寻找到真实的感情，而不是被舞台束缚，而站在阳光中，象征光想要把希望留下。

野蛮而残破的场景，罩衣被随意悬挂，代表规则被打破后，新的舞台会以一种前所未有的混乱姿态现身。

音乐变得轻盈明快，隆重而充满希望，原来刚才的网格是类似铁轨的形状，华恋站在铁轨上，一脸震撼，望着火车驶来的声音，似乎在迎接什么。象征新的未知的事物即将来到，华恋面对没有画面的火车，脸上还是迷茫的表情，代表新的舞台来势迅猛，甚至没有让她反应过来。

神乐光说了永别之后，华恋就连同模型一起被撞飞了，番茄也再次爆裂。在动画中，华恋和神乐光是被红色的星星从塔上撞飞，而这里是被代表未知的火车伴随着 positionzero的模型撞飞，象征单纯的环境下被目标伤害，复杂的环境下被未知伤害，伤害的程度更深，伤害的来源更广，番茄爆裂象征着舞台少女虽然打破了剧本的束缚，但是失去了规则的保护，代表着演员的牺牲，象征着成熟果实的落叶归根，散发养分。

神乐光说永别了，代表她是这场慌忙得来不及细细体验的战斗的主导者，她知道一切的真相，作为一个前辈的形象，对于华恋的迷茫发出质问，华恋似乎是那个面对新世界没有准备的晚辈，对于一切都没有实感，扮演着一个被动者的形象，然而这样的形象是不是真相，还要等待我们看完全部的剧场版剧情才能揭晓答案。

神乐光冷漠的主导者这场单方面优势的对决，代表冷酷的行为对于华恋的伤害，也象征神乐光以新舞台为幌子，重新伤害了华恋。

``` 开幕结束 ```

校园的场景掠过，华恋看着光恋两人的合照，光的配文是“我们已经在舞台上”，象征着舞台的定义十分宽广，在入学前就是人生的舞台，在即将毕业时，也象征即将改换舞台。

四名新生代表未来的希望，而华恋代表新生的挑战，象征华恋即将面临毕业的抉择，又是一个新学年，相信看过全部动画的人，在这里一定有所感触，新旧交替之时，就是矛盾爆发之时。

镜头转移到99期生，办公室里，A班众人都说出了自己的进路意向，分别表现了各自的个性，期间穿插过学院历史的照片，象征这是历史时刻的谈话，这里的一个小小决定，就能够影响今后人生的道路。

天堂真矢为了更高更远的舞台，傲然挑战新国立升学志愿，象征勇往直前的强者。

克洛迪娜在圣翔祭上的表演被烈焰剧场的人看中，主动邀请她前往参加考试，象征着主动脱离和天堂的较量。

露崎真昼也想报考新国立。

花柳香子只想继承千华流，每个人物的叙述都配合一段动画表现个性。

双叶为了挑战自己的可能，也想要报考新国立，她受到身边优秀同学的感召，决定奋力一搏。这是自克洛迪娜和天堂之后的又一个关系的疑点，要好的两人为了自己的前途分开，追寻自己的梦想，难道在圣翔这么刻骨铭心的一段时光就要这样抛之脑后了吗。

大场奈奈对于舞台和幕后的抉择不是很明确，但是很想享受舞台的快乐，象征留恋和美好。

纯那说自己要启航，但是已经找到人生意义的纯那还说自己要启航，意义存疑，而接下来，纯那的决定是这样的：

纯那对于大学的渴望，和对于文学的热爱，盖过了舞台的理想，想要考取大学。第一个放弃本行的决定出现，这对于纯那的沉没成本是巨大的，升学和梦想的矛盾再次凸显，而纯那在动画版中说自己因为舞台找到了新生，还对过去努力学习的自己感到遗憾，态度明显是积极拥抱舞台的，在这里放弃与舞台少女的精神不符。

``` 进路调查结束，光恋回忆开始 ```

13年前，那是光恋两人在日本上幼儿园的时候，神乐光对华恋大大方方的打招呼，华恋害羞的躲在母亲身后，象征神乐光从小开始的演艺才能，幼儿园里华恋对集体的逃避，象征华恋童年时期的孤僻，也与之后的开朗形成对比。

两人被各自家长带回家的时候，分别在不同时候往后看了一眼，代表对对方的喜爱，也象征错过对方。

光恋两人在节奏课堂上的互动拉近了两人的关系，伴随着宏大的音乐，两人逐渐亲密，一切都似乎要成为闺蜜的感觉。

神乐光着急和华恋见面，但是看见华恋在和其他人玩耍，顿时感到失落，止步不前，看着自己的白先生，似乎在想着什么。象征神乐光虽然表面上落落大方，但其实对于自己的魅力有深深的怀疑，看见自己喜欢的人有离开自己的倾向，甚至不敢直接上前搭话，而是看向自己的精神寄托，来转移注意力。

最终还是华恋主动向神乐光搭话，华恋很高兴见到神乐光，两人一起走在回家的道路上，过了过了一会儿，神乐光才向华恋开口，先是用华恋的游戏机作为话题的引子，然后搬出了明天一起去看什么的邀请，在神乐光的急切心情下，甚至邀请的话术都有一些僵硬，华恋只是说着不知道之类的话，此时的华恋对于舞台还没有概念，只是看着神乐光如此渴望，华恋才勉强答应了。

神乐光给了华恋一封信，这封信也是贯穿全剧的线索，里面装的什么，会在下一次回忆的剧情中交代，神乐光说明天去接华恋，代表这个时候神乐光是交往当中主动的一方，是她主动邀请华恋走入舞台剧的世界的。

画面回到现实，华恋无精打采趴在桌子上，代表神乐光的离去让华恋很失落，也代表神乐光不辞而别的不负责任。

奈奈用童稚的声音配合巴洛克舞曲演唱，代表只有返璞归真、思维跳出框架、才能做到生存。

高架桥上这一出戏码，直接将动物的野性，通过躺倒的众人、得胜的奈奈、被逼迫到车头的天堂克洛，分别所处的状态，表现了出来，那就是对于学校舞台规则的破坏，对生物原始野性的激发。

高架桥上的奈奈重复问出了我们去向的问题，虽然字面意思没有变，但是和一开始对于众人的拷问有了意境的区别，一开始的拷问是富含哲理的开场词，现在的拷问是配合残酷现实的逼问，这是在逼迫她们快速想出答案。

克洛不理解奈奈的意思，过了好几秒，天堂才反应过来，回应自己的价值就是满足看客的期待，奈奈依然无精打采看着天堂，看来，这个答案不是奈奈想要的。

火车重新进入隧道，奈奈和天堂逐渐靠近，相互蓄积危险的气息，短暂的4/4拍转瞬即逝，以一节7/8拍的间奏衔接，配合克洛的挑衅，将克洛和奈奈擦肩而过的情节表现为一个小高潮，又重新返回4/4拍，变成天堂和奈奈的对决，好似克洛就是一个顺手打败的过客而已。

奈奈重复着想要活下去的台词，将剑插在了T位，配合一段滑音，皆杀revue就这样结束了，此时，奈奈完全站在了T位的位置，相比一开始的站位偏差，这时更有一种霸道和自信的气质。

看到这里，可以感受到奈奈这个角色的定位模糊，既像是一场血腥revue的导演，又像是一个恨铁不成钢的前辈，全曲的主题就是揭示社会的真相，打破自我麻醉的幻想。

奈奈说好像是喝了很烈的酒一样，表现的就是这个意思，模糊的身份定位，让奈奈启发他人的的过程变得苦涩和无奈，明知道这个revue是和同学们的告别，但为了让同学们接受自己的意志，她必须割下同学的纽扣，这种撕裂感和违心感，让奈奈感觉到自己做出了牺牲，就像苦酒一样难以下咽。

同时奈奈的revue也是一种预示，她已经传达了戏剧的意义，初步完成了自己作为先知者的使命，这样的行为，让奈奈意识到了自己的勇敢，就像有酒壮胆似的。

做出抉择的无奈之苦闷，和传达意志的责任之履行，混合在一起，成为奈奈现在的感受，就像酒一样复杂而激烈。

纯那不理解奈奈的心情，只觉得这种表达不符合好青年的价值观，奈奈无力解释，只是用舞台戏法营造出了纯那被割喉的效果，代表奈奈对于这位旧友既痛恨又怜爱又无奈的复杂感情，在一片血雨中，众人感受着这观念的剧变带来的痛苦，沐浴在苦涩的、代表失败的血液中。

作为唯一一个没有被摘下纽扣的天堂，提醒众人这只是舞台装置，稍稍平复了一些气氛，也算是对于观众情绪的照顾，对于剧情节奏的缓和。

奈奈第三次重复了那句话，但是她说“我们”已经死掉了，配合没有音乐的音效，这里的意境相比前两次，多了一丝迷茫、少了一层强势，似乎面对不成器的同学们，奈奈也开始怀疑自己这么做究竟有没有意义，这些人到底能不能理解自己的意思，于是只好做出最终的警告，这绝望的话语，象征着对于旧日同窗情谊的最终道别。

奈奈的火车头快速前行，将众人拉开了距离，代表奈奈还有更重要的事情要去做，也代表对于同学们的失望，失去了奈奈的动力车厢，这一节列车也很快停下来了，众人相顾无言，香子品尝了一滴血液，说“amai”不知道是用这道具血液的甜美麻痹自己，还是用“天真”这个词形容包括自己在内的众人。

``` 皆杀的revue结束 ```

给华恋的信件展开，那是一张演出的海报，光给邀请华恋去的地方正式揭晓，剧场版中有光恋两人的多段回忆，是对于两人关系的补充说明。

华恋双眼呆滞看着海报和小姨和母亲希望通过看戏改变华恋怕生性格的情节，强调着此时华恋的性格，家长的态度是“死马当做活马医”“乱拳打死老师傅”的模糊态度。

门铃按响，华恋前往开门，母亲在此时表露出了想要打扮华恋的愿望，是下一次回忆的伏笔。

门外是一片红色，这是代表罪恶的星星的颜色，华恋站在门口发愣，代表对于未来的恐惧，踌躇不前，在剧本的残酷真相下，被赋予了更加复杂的意义。

开幕的铃声响过，舞台上出现的是被红色布匹覆盖的很多银色模型	，代表摘星之路的代价之一就是前辈的隐退。

台阶上躺倒的神乐光，其实是华恋视角中的目标，青年状态的神乐光参与画面的构成，代表摘星之路贯穿整个18岁以前的时光。
依然是华恋的罩衣披挂在旁边的剑上，与以往不同的意义是，此时神乐光闪耀的养料之一就是华恋。

紧随其后的一个镜头更清楚地表明了寓意，红色的幕布如同触手一般，抓着银色模型，一副吸取养料的阵仗，在中间的星星，就是养料供给的中心，如果说动画版最终结局是两人打破剧本，创造自我，那么这个画面就代表神乐光将华恋榨取，独自闪耀。

这一幕和剧情配合起来，有一种反差的美感。两人的两小无猜和神乐光辣手摧花的强烈对比反差。

华恋对于这一切都还不理解，神乐光解释说，舞台上什么奇迹都有可能发生，象征舞台其实是闪耀和危险并存的，此消彼长，福祸相依。

神乐光强调舞台对于个人的塑造，对于荣耀渴望的满足，画面却搭配光昏睡在华恋怀里的画面，象征着神乐光杀伐之气深重，一边如同大姐姐一般给别人介绍舞台，一边又依赖于华恋的情绪价值。

观众席上，华恋站在座椅前，被戏剧深深吸引，光站在灯光的阴影中，象征神乐光躲在暗处的状态，暗示华恋陷入陷阱。

两人的位置互换，但是都手握红色幕布，象征两人的命运被舞台连接。

舞台中心青年的两人脚下延伸出两条幕布被童年的两人拿住，象征童年的光恋是长大后自己的养料，青年的光恋是小时候自己的目标和陷阱，而舞台上青年的形象也象征着辈分年长的女神，散发自己的光芒，启迪后人的愿望，同时将轮回转嫁到了小辈身上。

神乐光抓紧了幕布面露忧色，也代表她对剧本的理解更加深刻，但是下一秒被天真的华恋安慰，从而真正有了登上舞台的勇气。

神乐光受到鼓励，结下了约定，点头答应之时，镜头却突然切换到青年沉睡的自己身上，象征神乐光的罪恶和悲剧人生。

青年的光慢慢醒来，象征华恋的勇气和力量让神乐光也感受到了。

表演结束，这是starlight剧本中的剧场版剧本，和原版相比，删减了一些情节，但正是这样，才有两人的童真，才有华恋的入坑。

两人在合照留念，华恋紧张的表情说明剧场对她幼小心灵的刺激，也象征舞台的闪耀之力量，让无数观众心向往之。

而这张照片，也是光给华恋手机发送的那张配有“我们已经在舞台上”文字的图片，代表舞台上的人生，最早在作为观众的时候，就已经开始了。
	
冷酷、寂静的列车内，奈奈向华恋提示着舞台梦想的真谛。

而华恋作为唯一一个在日本但是没有被皆杀的人，象征对于梦想的反思，对于初心的提振，是奈奈心中更接近正确答案的做法，所以只有心态不浮躁，才有在狂野大舞台上胜利的资格。

奈奈报菜名来表达人生道路的冷酷和人本身的浮躁，这种浮躁虽说是覆水难收的残酷规则所致，但是也是少女们最隐蔽的陷阱。

奈奈认为只有华恋没有被表象蒙蔽，只有华恋才是最有资格前往下一个人生舞台的人。

列车从闹市区直接驶进沙漠，象征返璞归真，去伪存真后的世界，其实非常简单。

奈奈告诫华恋一定要找到自己的舞台，这其实是对于剧本的淡化，对于舞台依赖的摆脱，这也是奈奈认为的生存之道，如果只是别人的提线木偶，那么舞台少女们就会死掉，作为没有灵魂的空壳游荡。

``` 列车到达沙漠 ```

沙漠中如同孤岛一般的站台上，奈奈下车了，象征她对舞台理解的局限性，华恋依然具有不确定性，所以能够继续前行。

而奈奈所说的自己必须回去，就是她作为母亲的关爱必须发挥出来，对于纯那的决断必须做出，华恋拥有理想的力量，拥有继续前行的可能，但是自己不是华恋，必须回归同学们中间，在集体中获得力量。

华恋在车厢中看着代表两人约定的信件，努力思索着只属于自己的舞台是什么意思，光恋两人的线索贯穿全剧。

回到101期圣翔祭誓师会，真井慌忙地在奔跑，在不断寻找什么，宴会上的食品丰富多样，但是唯独没有香蕉套餐，象征虽然是99期的毕业晚会，但是99期生的身影却不见，主场让给了新生，99期生各怀心事，为接下来的剧情做铺垫。

纯那独自坐在凉亭下，真昼也呆在走廊处，双叶望着集会中心发愣，香子靠在摩托旁沉思，众人都没有从皆杀中回过神来，而沉思，所做的思想准备，就是接下来各自revue的理由。

天堂作为受到冲击最小的人，显得很享受毕业晚会，而克洛在教室内望向天堂，为皆杀中自己的淘汰感到不解，为天堂的幸存感到嫉妒，她看向手机，不断思考这段邀请词的真正含义。

这一段心理描写为之后的revue打下基础

准备室内，真井走向面色凝重的雨宫，安慰没有写完剧本的她，原来刚才焦急的寻找，就是指这个，雨宫并没有因为收到安慰感到安心，反而因为没有完成任务而自责。  

真井就提示雨宫她们之间的约定，在誓师大会之前？？，看样子应该是写好剧本，真井鼓励说现在不得不前进了，别人都在等着，就算是未完成的剧本，都必须展示。

这一段台词有几层意思，对于规则的遵守，是参与舞台的资格，截止日期的冷酷，代表覆水难收的人生，他人的期待，代表舞台的养料，未完成剧本都要展示，代表人生的仓促和守约的重要性。

101期剧本就这样赶鸭子上架了，且看她们如何回应舆论，一开始，真井并没有主动提出剧本的缺陷，也没有一味道歉，这不是对于错误的逃避，而是为了施展语言技巧，故意让别人指出。

纯那是最固守成规的人，最先指出剧本未完成，真井面不改色，直接承认，众人交头接耳，议论纷纷，真井没有马上打断，而是任其发酵，等到情绪到位了，就用一声怪叫吸引众人的注意，同学们都被震惊，给接下来的表演腾出注意力。

然后用对于未来的恐惧，为没写完剧本开脱，但是真井直接调动众人的积极性，期望共同创造剧本，这一顿操作直接让危机被化解，而雨宫这个时候才出来道歉，显然就不会再受到指责，还收到同学们读词的鼓励，香子却暗自离去，代表她并不喜欢这样的解释。

众人的齐心协力下，摘星塔被拉起，望着这座塔，大家都接受了这个剧本，天堂说：我们已经在舞台之上了，一片欢乐祥和的景象出现，似乎领悟到了人生如戏道理的众人，已经有资格站在全新舞台上了。

这一段里，真井用高超的技巧，化解了剧本未完成的危机，展现了两人的羁绊和帮扶，天堂领悟到了这种解释的含义是人生如戏的意思，而这演戏，就是真井的能力，刚才还不屑一顾的香子，都停下来感叹舞台的美丽。

新的戏剧，在舞台还没准备好的时候就上演了，克洛迪娜都感到惊讶，代表这种戏码却是是前所未有的，也是打动人心的，和人生结合起来的戏剧，比干瘪的剧本，更能得到认同。

奈奈站在摘星塔的背面，作为幕后者的身份，为这一切喝无声的彩。


``` 誓师会结束 ```


由奈奈顺滑转移到由蔬菜水果组成的长颈鹿身上的镜头象征了奈奈就是作为养料的人，支持着众人，同时也是亦师亦友的人物，引导着众人。

列车中间有很多前后方向交错的T模型，众人躺在列车上，代表旧的自己已经彻底死亡，而新生的自己站在血泊当中，看着倒在地上的自己，象征焕然新生。

奈奈的教导，也可以说是社会的教导，让她们感受到了险恶，知道了自己的不足，了解了残酷的真相，有了自我怀疑，就有了适应新环境的条件。

纯那看着代表自己的过去尸体，依然对于奈奈所说的了断有所不解。

代表入场券的番茄滚到天堂脚下，听着长颈鹿关于舞台的解释。

长颈鹿所说的自己就是舞台少女的食粮，代表舞台少女虽然为舞台奉献了青春，但是自己的命运已经和舞台绑定，不得不依靠舞台为生。

需要注意的是，燃料这个词语在动画版中出现的意思是，成功的舞台少女以失败的舞台少女为燃料，牺牲了他人成就了自己，而这里长颈鹿也是燃料，代表的意思是，除了其他失败的人以外，舞台上的一切，包括自己的才华和能力，自己付出的汗水，都是支撑自己表演下去的必须条件，甚至必须要献祭自己的过往，才能获得生存的权力。

以红色调为主的蔬菜，组成长颈鹿的头，在黑暗的环境下显得很阴森恐怖，将新舞台的诡异表现出来，配合几个五光十色，绚丽多彩的镜头，代表新的舞台包罗万象，险象环生，满地荆棘。

众人吃下代表入场券的番茄，象征她们已经尝试彻底抛弃过去的自己，成为新舞台上那个全新的自己，失去了青春期的无限可能，开始为社会的奔波而将自己塑造成奇怪的形状。

三个T模型，中间的箭头朝前，两边的箭头朝后，代表把过去的经历和成长作为养料，才能继续前进，而过去和现在的方向是大相径庭的，在这个节点，自己必须走向一条全新的道路，向着陌生的方向。

这一段用意象化的表达，将学生在面临毕业抉择时候的矛盾心理非常巧妙地展现了出来，并将社会的残酷真相用非常多象征意义的镜头暗示了出来，非常生动形象，但是更加值得称赞的是，真实。

而台词给出的解释是“公演者，还差两位”，代表华恋和神乐光还没有出现，她们暂且处于寻找前路的过程中，她们的养料还没有被吸收，她们T模型指向的方向和众人不一致，还没有获得入场券。

背景音轨中也出现了华恋小时候挽留神乐光的声音，将剧情再次拉入两人的回忆当中。这次的回忆比前两次都要长，会更加细致展现两个人分别在两地的心理状态。


``` 毕业晚会结束 ```

这一段是两人分别时的回忆，表达了华恋的愿望是两个人一起闪耀，而神乐光的愿望是成为star，而神乐光离开的理由是随便找的一句“华恋总是撒娇”。

小时候在滑滑梯的互动在动画版的第八话中第一次出现，而其深层次的内涵终于能在剧场版中用长篇幅的剧情表现，得以让我们观众一饱眼福。

此处的东京塔随着镜头不断上升，塔身的形状也变得越来越抽象，象征回忆的淡化，可见接下来是要描写光恋两人负面情绪的剧情。

在某一天一定会背靠着背，去看那最亮的星，明明做了这样的约定，分别前的这条路对两个人来说，已经走了足够久的时间，足够久的时间。

两人在阶梯上的场景被切掉，取而代之的是一处金碧辉煌的夜总会，香子坐在吧台上双手撑着桌面，慵懒的说“烦人”，而客人打扮的双叶愣了一下，对于香子这种反应感到诧异，而这种情感的原因也即将在接下来的剧情揭晓。

在revue最开始的那座小屋内，挂着能面，象征这场戏剧是能剧的形式，而从夜总会的镜头开始，音乐变得暧昧，背景有些滑稽，风格的巨大转变代表已经进入能剧的穿插剧目：狂言。能剧是注重舞蹈和音乐的配合，庄严悲壮的戏剧，而狂言是穿插在能剧当中，注重对白、幽默和讽刺的运用，用轻松欢乐的氛围，点出具有关键矛盾的剧情，用巧妙地文戏，配合整个能剧表现主旨的插曲。因此接下来一段剧情虽然看上去也许有些无聊，但其实是理解这场revue的关键时刻，是对于后半段感情的铺垫，请各位观众无比仔细欣赏。
