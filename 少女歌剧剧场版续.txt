大家好，上次我们说到，奈奈打断纯那虚伪吟唱，脚推短刀要求切腹，纯那绝境之中唤醒自我，夺刀站起准备反击。


那么本期就是纯那和奈奈之间的最终决斗了。

接下来一个场景切换到满屏黄色烟雾的一处高台，台上有一把短刀，象征奈奈的威胁还没有解除，但是画面主题颜色由紫色变为黄色，说明纯那已经接近奈奈的核心，一转攻势了。

歌词开始，纯那一改之前用他人直言为自己鼓气的作风，转而直白的说出了自己的诉求，一切才刚刚开始，怎么能停下脚步，这是她的觉醒，她不愿放弃这么多年的训练和积累。

察觉到纯那并不会如自己的意愿放弃前进，奈奈要求纯那把刀还给自己，而纯那一直以来用弓作战和现在用刀作战，象征她从动画第二集当中躲躲藏藏使阴招的人，变成现在用近战武器，正面对抗的勇士。

而第二句歌词里面的名言也变成了非常相称于现在纯那心情的一句话了，这说明纯那在找回梦想之后并没有抛弃自己以前最喜欢的名言，而是将名言转化为了自己的力量，与这句歌词同时出现的奈奈歌词还是劝说纯那就此投降，歌词中的烟雾即是指现在场景中代表自己的黄色浓烟，也是对于纯那模糊前路的贬义称呼，但已经由刚才的审判姿态转为被动了。

纯那到底是弓兵，使用长刀被奈奈打得节节后退，甚至眼睛都被打掉了，在奈奈唱歌词的时候，上方的短刀通通指向了纯那，难道纯那要就此被压制吗？

从天空上降下一把刀，刀刃指着纯那，这代表纯那必须从继续战斗和投降之间做出选择。

而奈奈还在重复纯那的缺点，全然落后于纯那现在真正的境界，已经暗示了纯那即将突破自己，她重新拿起了长刀，歌词也正好唱到：“有种就来杀了我”，挑衅和自信的气势凸显。

奈奈握紧手中的刀，返回叫纯那不要继续打斗了，她开始隐隐害怕纯那的决心和勇气了，告诉纯那：“已经该落幕了”，这是她对于自己心中规则的机械运用，是对于纯那适应能力、自我调节能力的轻视。

而“我的纯那才不是这样的角色，你不是我的纯那”，也代表奈奈对于纯那的武断定义让自己做出了错误判断，以前的纯那确实是自卑没有勇气的人，但奈奈一直都是比较极端……比较看重规律，逃避改变的。

在动画版中，奈奈接受了舞台必须向前发展的观点，在皆杀中，因为自己实力强劲所以让其他人接受了自己的观点，而在现在，奈奈又变成固守成规的人了，始终认为纯那无可救药，是个书呆子。可以看见这个人物的角色在不断变化，她不愧为妈妈类的角色，在各个地方推动着剧情的发展，最先挑起矛盾，却最后幡然醒悟。

纯那重新站起来后把奈奈打得后退，自己的脚也踏上了T位的一角，代表接近胜利。而舞台颜色重归正常，代表奈奈的压倒性优势已经消失，现在谁的优势也不是，是回归常规的舞台。

聚光灯照亮纯那的位置，她否认了奈奈的定论，向前继续走了一步、打出一刀，周围的短刀吊饰被一刀两断，代表悬在她头顶的达摩克里斯之剑消失了，纯那结束了自己畏畏缩缩、逃避现实的时光。

而此时舞台也断裂为两半，这个情景在动画第三集和第十二集中分别出现。第三话的天堂为了表现出自己和华恋实力的差距，将自己的平台抬高，而出现很多断层，是为了凸显出华恋和成功之间的差距。

第十二集中，是为了展示弗洛拉和克莱尔之间悲剧般的隔阂，最终被摘星塔连接起来说明跳出剧本的勇气和力量。

而这里的纯那斩断舞台后，从自己站的一部分跳到奈奈的一部分了，舞台的断面是紫色的，但是这次不是奈奈斩断的而是自己斩断的，代表和过去的自己一刀两断，而连带着斩断的还有舞台上的一个青蛙标志，说明原来的自己是被奈奈烙印上她最喜欢青蛙的标签人，而现在不仅和过去的自己一刀两断，而且撕掉了自己被贴着的标签。

摘星塔的照明灯灭掉，这盏灯是随着奈奈的到来而亮起，如今随着纯那的崛起而熄灭。

纯那挥剑一砍，瞬时间地动山摇，奈奈的纽扣缓缓升天。

在一处紫色砖块铺设的分岔路前，两人背对站立，纯那说：“那一定就在下一个舞台，不畏漫漫前路，深处渴求的手,哪个星见纯那成为主角的舞台”奈奈说：“我的再演可能已经结束了，我也该去找只属于我的下一个舞台了，但是总有一天”，纯那回应：“总有一天站在新的舞台上，一起”。

两人互相道别，奈奈的照片和相机沉在水里，象征奈奈已经放下了回忆，而那张被奈奈劈开的照片也慢慢粘合。

两人合唱：“总有一天会在舞台上再度相会”，奈奈也流下眼泪，说果然好耀眼，纯那向后稍稍看了一眼，但随即转过视线，继续坚定向前，奈奈也擦干眼泪，继续向前，留下原地被削成两半的摘星塔。

两人的和解和友情就是本场revue的落幕了，两人在经过激烈的战斗后，都意识到了自己的错误和不足，也重新认识了对方的价值和闪耀。两人互相道别，表示自己不再恨对方，也不再拘泥于过去，而是希望在未来的舞台上再次相见。

两人在战斗中都突破了自己的极限，找到了属于自己的星和舞台。纯那从一个自卑、逃避、模仿他人的弓兵，变成了一个自信、勇敢、创造自己的刀兵。奈奈从一个固执、自私、压制他人的刀兵，变成了一个宽容、无私、鼓励他人的弓兵。

在作品中，弓兵和刀兵是两种不同的武器和战斗风格，也象征着两种不同的人格和价值观。弓兵代表着远程、防御、支援、团队、谦虚、合作等特点，而刀兵代表着近战、进攻、独立、个人、自信、竞争等特点。

在剧中，奈奈一开始是一个刀兵，她用自己的刀压制其他人，强行执行自己的规则和理想，不顾他人的感受和意愿。她认为自己是最优秀的舞台女演员，也是最合适的TopStar，她想要保持舞台的不变和完美，不愿意接受任何改变和挑战。她对纯那有着很深的感情，但也有着很强的占有欲，她想要纯那永远做自己的搭档和朋友，不允许纯那有自己的梦想和选择。

在与纯那的战斗中，奈奈逐渐意识到了自己的错误和不足，也重新认识了纯那的价值和闪耀。她看到了纯那从一个自卑、逃避、模仿他人的弓兵，变成了一个自信、勇敢、创造自己的刀兵。她感受到了纯那对于舞台的热爱和期待，也体会到了纯那对于自己的友情和尊重。她最终放下了自己的刀，也放下了自己的固执和自私，接受了纯那的挑战和胜利，也接受了舞台的变化和发展。

``` 奈奈和纯那的终战结束 ```

大家好，上期我们说到纯那觉醒自我，奋力反击，最终两人心心相惜，相忘于江湖。这次来到一小段光恋的回忆，这段也最终彻底揭开两人的往事。

返回华恋的回忆，东京的街道上，华恋独自放学回到家里，从桌面上的信件里，看见了一个人的照片，虽然只有下半身，但是还是让华恋一眼认出了这个人自己见过。

她对神乐光念念不忘，躺在床上，终于忍不住拿出手机查询神乐光的信息，尽管她一再强迫自己不要查询，手指还是不争气的按下了检索按钮，在英国王立戏剧的官网找到了神乐光的入学信息。

我不知道华恋此刻的心情是怎样的，或许是激动，或许是失落，或许是羡慕，或许是恨。她看着屏幕上那个熟悉而陌生的名字，感觉自己的心被一把刀刺了一下，想起以前那些日子，他们的相遇，他们的相知，他们的相爱，还有他们的分离。回想起神乐光的离去，那已经不知道是多久以前的事情了。这段时间被兴奋和恐惧折磨的华恋，终于忍不住了，得知了神乐光的消息，华恋心里的一块石头落地了，她激动于对方还没有忘记约定；失落于自己依然不如对方的地位；羡慕于对方考上了最难考上的学校；恨两人之间的相遇让这一切感情无中生有；闯入自己的生活折磨自己；也庆幸自己有神乐光走入自己人生，带来精彩。

她想给他打电话，想听听他的声音，想问问他在哪里，想告诉他她有多想念他。但是她没有勇气，她怕听到的是冷漠的拒绝，或者更糟的沉默。她怕自己打破了这份美好的回忆，怕自己失去了这份唯一的希望。她只能默默地关掉手机，把它放在枕头下，抱紧自己，闭上眼睛，假装他就在身边。

而考上圣翔后，神乐光的转入让华恋惊喜若狂，一切的感情都尘埃落定，两人的相见就是无声的交流，华恋眼神中充满爱意，在接神乐光去寝室的过程中，有一句没一句的说着这些年自己的感情，隐瞒着自己知情的事实，赞美着神乐光，虽然仿佛要把一切话语都说出来，最后却只是用手指着皇冠的头饰，表示两人的约定依然有效。

而此刻出现了华恋罩衣被挂在剑上，朝左追逐着光的景象，既是神乐光对于华恋的伤害、弗洛拉为克莱尔做的牺牲，也是真昼所无法原谅的行为、让华恋卷入剧本、却用最坏的方式演出。

尽管有这么多难受的事情，华恋却从来不会生出“多么希望不曾有这样的体验”的感觉，神乐光却正好相反。

而华恋说“这是我私下定下的规矩，可能也没什么要道歉的”，像是对于事实的阐述，更像是对于自己的自我安慰，他明知道光也很喜欢舞台，她们的重逢注定是命运的延续，约定的履行，但依然不肯定事情会不会如自己期望的那样发展，神乐光的毅然离去，让华恋记忆犹新，被伤害的恐惧让她牵肠挂肚，如梦魇缠身。



华恋相信着美好的明天、两人的共演一定会到来，神乐光却站在树荫里，微微颔首，华恋是重逢的喜悦，神乐光是遭遇挫折的自卑，她们的故事，将在剧场版最后一幕揭晓。

华恋的煎熬如香子一般孤独，如双叶一般炙热，如纯那一般渴望，如奈奈一般隐忍，如天堂一般高傲，如克洛一般敏感，再包含舞台的冲击和剧本的折磨，共同组成了华恋复杂的感情，让她对于光期盼又害怕。

一处通往东京塔底部的列车轨道，华恋走在枕木上，缓缓向前，塔尖上两颗星星，代表看似很近，其实是遥望天河的距离。

这一小段的回忆虽然很短，却包含了华恋对神乐光的所有情感，她既想忘记那些伤痛，又想留住那些美好，她既想追寻他的脚步，又想保持他的距离，她既想和他共同登上舞台，又想让他远离剧本的诅咒。她不知道自己该怎么做，她只能在心里默默地祈祷，希望有一天，他们能够真正地相拥，相视，相爱。

``` 光恋最后一段回忆结束 ```


好，我们继续，上次说到华恋和光最后一段回忆结束了，光恋两人的故事也逐渐接近。这一集是克洛迪娜和天堂真矢的决斗，她们又究竟会摩擦出什么火花呢？

两人的准备室是化妆间，这个设计和香子的赌屋、奈奈的军服、露崎真昼的运动场一样，都体现了人物的性格特点，而化妆间说明这两个位居第一第二的演员，对于本职工作非常看重，对于化妆十分愿意花心思打理，她们对于舞台的重视程度其实是高于其他舞台少女的，真昼喜欢可爱的动物和体育运动，奈奈有军人般的执着、武士般的情结，香子宠溺成性，喜欢耍小聪明、喜欢赌气，而她们是要在舞台上展现完美的自己。

两人在棋盘上玩百兽棋，天堂在这个时候问“我的小鸡去哪儿了”，同时表现出诧异的表情，坐在对面的克洛就显得古灵精怪了，气势上甚至隐隐压了天堂一头，这样的对比说明克洛是非常狡猾的人，对于事物的观察细致入微，思维很缜密，百兽棋的水平强悍就是辅证，包括在皆杀之前对于露崎真昼的调笑以及很多地方，都说明这一点，天堂真矢拥有最高超的表演技巧，但是对于克洛迪娜这种死缠烂打的行为还是有些难以应付，多次为这个挑战者的轻浮行为感到困扰，而这种对比即将在接下来的对决中体现出来。

克洛在棋盘对决的时候说：“本以为我早就将舞台作为人生的目标了，我却没能去向下一个舞台，逐渐腐朽死去”，这句话是克洛对于自己童星身份的疑惑，对于被天堂打败的不甘，暗示是天堂的到来才让骄傲的克洛遭遇挫折，而好胜心强的克洛绝对不会放任有人在自己前面。

在克洛说这段话的时候，镜头对准自己的时候是镜子中的自己，对准天堂的时候一开始还是镜子中的影像，下棋后，对准天堂的时候的却是镜子中反射的天堂以及本人，代表这段话是虚幻的克洛对于天堂和天堂背后代表的实力的挑战。以虚抗实，以影盖面，是接下来克洛的主要战略指导思想。

克洛想要超越天堂，但是此刻只能追着天堂的影子跑，而没有真正的超越天堂，此刻的首席，在克洛眼中，还是在镜子中扑朔迷离的形象，可望而不可及，但这只是表面上的状况，克洛在棋盘上战胜天堂多次，就说明她并不是一味追逐着天堂，而棋局象征着智谋和野心，克洛的一转攻势将马上到来。

镜头再一转，百兽棋棋盘上的小鸡又回到了棋盘上，而归还小鸡的动作，代表克洛心情的变化，结合她的台词，我们知道，克洛接下来说的，就是对于自己被压在人下的不甘之后，克洛的具体行动了，果不其然，克洛就在这个时候说：“所以我要重生，为了和你做一个了断，去往下一个舞台”。

说话的时候，两人的棋局是两只小鸡面对面针锋相对，代表克洛对抗天堂的正面战场，而就在这个竞争白热化的关键时刻，克洛移动了左下角的車，車这个棋子的灵活性代表不可预测的神秘行动，关键时刻的杀招，暗地里的阴招。

面对克洛的阴招，天堂的应对策略是向前移动王，也就是利用自己引以为傲的实力，用压倒性的优势，抵消克洛的旁门左道带来的影响。正面对抗，代表天堂的傲慢慢性格，以及对于实力的盲目崇拜，和对于阴招的不屑一顾，一个傲然自赏的天堂形象跃然纸上，棋盘之上的战斗和较量反映着她们的心理活动。

镜头一转，一颗車的棋子靠在纽扣上，这正是刚才克洛下的一步暗棋，代表克洛将在接下来的对决当中，于阴暗之处，埋下伏笔，而在这样的暗棋镜头之下，天堂的其他动作显得没有那么紧要了，这说明在接下来的对决中，克洛的技巧，将会发挥四两拨千斤的作用。

天堂听言，笑着说欢迎登上舞台，这和她的一贯作风并没有区别，但是克洛的回应是：“你还真的是弱啊”，这里的你很弱这句话，和刚才的弱有一点不同，刚才是克洛挑逗天堂之后对天堂的迟钝加以调笑，这里的你很弱，代表克洛对于天堂正人君子般的气质对于战场技巧的重视程度不够的结论，克洛认为天堂是被自己引以为傲的实力和荣耀所蒙蔽了双眼，不敢面对对手的阴招，而阴招的破坏性依然存在，这样的轻敌行为，让克洛觉得自己已经占尽先机，找出了对手的弱点，对决是十拿九稳，至少也是胜率大增，她就差窃喜的说出“兵不厌诈”并摆出诡计得逞的坏笑表情了。

同时棋盘上的棋子变成天堂一方棋子全部上阵，而克洛的棋子只剩一个王孤军作战。这说明天堂在面对克洛的小心思的时候，没有做到知己知彼，调整战术，而是一条路走到黑，上了所有的棋子，展现了自己全部的实力，依然试图硬碰硬，用正面压制的方法对抗，克洛刚才还有几颗棋子是在棋盘之上的，现在却一点都没有了，这不是天堂实力强劲，力压群雄的象征，而是克洛阴谋已经酝酿完成，在棋局表面看来是独木难支，其实早就准备好了盘外招，在最终时刻使用，让天堂遭遇滑铁卢。

同时整个场景都设计在化妆间，说明这是一场真真假假，虚实难辨的对决，智斗为主，武斗为辅，拥有最强实力的两人，都是勇者，但是勇者相遇的时候，智者将取得胜利。



``` 天堂克洛化妆间准备完毕 ```
